About NBAGrid

NBAGrid is a daily puzzle game where you try to find NBA players that match specific criteria. Each day presents a new grid with different combinations of player attributes to discover.

Created by Nils Brinkmann, this game is inspired by popular grid-based puzzle game GeoGrid

The code for this game is open source and available on GitHub.
